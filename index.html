Welcome to Language Centre Onine - Flash Content Review

Flash content will no longer be supported in the Language Centre by the end of this term, this because major browsers have been slowly pulling support for years now, and Adobe itself has announced that it would be stopping development and support in December 2020. This means that within the year, pretty much no mainstream browsers will be able to access Flash content on the Web. At the moment flash content is accessible via user selected activation of plug-ins, but this will not last. Support could be removed at any point and without notice.

If you would like to still be able to use any of the resources listed below please contact Nebojša Radić – (nr236@cam.ac.uk).
